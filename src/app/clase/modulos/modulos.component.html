Modulos

Los modulos son una forma de segmentar nuestro proyecto para poder derivar responsabilidades de manera independiente cada modulo podra importar, declarar, exportar y proveer.


Los modulos se pueden importar entre ellos y es importante no hacer una referencia ciclica, para eso tenemos el metodo de importancion por routing o por el mismo .module ... Es importante recordar que los modules pueden tener un hermano routing aunque no es indispensable
