React Form

React form forma parte de la libreria ReactiveFormsModule de angular y nos sirve para construir formularios de manera reactiva. Gracias a este modulo no dependemos de ngModel y nos facilita la capacidad de manejar valores, hacer validaciones ya sean sincronas o asincronas e incluso podemos hacer nuestras propias validaciones.


Es importante recordar que debemos determinar un etiqueta form que llevara el control de nuestro FormGroup y cada elemento tendra un formControl que se identificara respectivamente en el template con su formControlName
