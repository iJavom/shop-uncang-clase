Pipes

Los pipes son un caracteristica de angular que permite transformar y dar formato a los datos que estamos gestionando en la aplicacion. Los cambios se realizan en tiempo real y se dan gracias a los ciclos de vidas en su proceso de renderizacion.

la sintaxis de los pipes es la siguiente: abrimos nuestros interpolacion con las { { variable | pipe } } donde nosotros podremos modificar la visibilidad
Existen diversos tipos de pipes y lo podemos clasificar basicamente en 2... nativos de angular y personalizados

Pipes nativos de angular

Currency Pipe

El currency pipe sirve para transformar nuestros datos numericos en formatos de moneda
el pipe tiene un nombre de currency y le podemos a√±adir atributos... ejemplo personalizado: currency:'EUR'

Ejemplo:

Este es sin pipe: {{precioEmpanada}}

Con pipe: {{precioEmpanada|currency}}

{{ precioEmpanada | currency:'COP':'$':'1.2-16'}}

Decimal Pipe

El decimal pipe sirve para dar formato a los numericos (y en caso de haber decimales redondear)
la sintaxis es : number : digitos : locale

Ejemplo:

Sin pipe : {{numeroPi}}

Con pipe : {{numeroPi|number}}

{{ numeroPi | number:'1.2-2'}}

Percent pipe

El percent pipe se utiliza para formarte un numero como porcentaje donde se trabaja entre 0 y 1 siendo la sintaxis:
numero | percent

Ejemplo:

Un numero sin pipe: {{porcentajeEjemplo}}

Con pipe: {{porcentajeEjemplo | percent}}

{{ porcentajeEjemplo | percent:'2.2-2'}}

Uppercase pipe

El uppercase pipe sirve para convertir nuestro string en mayuscula

Ejemplo:

Sin pipe: {{ mensajeEjemplo }}

Con pipe: {{ mensajeEjemplo | uppercase}}

Lowercase pipe

El Lowercase pipe sirve para convertir el texto en minuscula

Ejemplo:

Sin pipe: {{ 'HoLa EsToY InTerCalAdO' }}

Con pipe: {{ 'HoLa EsToY InTerCalAdO' | lowercase }}

Titlecase pipe

El titlecase es un pipe que sirve para convertir la primera letra de cada palabra en mayuscula y el resto lo deja en minuscula

Ejemplo:

Sin pipe: {{ 'Un posible titulo va aca' }}

Con pipe: {{ 'Un posible titulo va aca' | titlecase }}

Slice pipe

El slice se utiliza para extraer una porcion de un string o un array y mostrarla en pantalla
su sintaxis es : slice : inicio : final?

Ejemplo:

Sin pipe: {{variableEjemplo }}

Con pipe: {{ variableEjemplo | slice : variableEjemplo.indexOf(filtro)}}

{{ variableEjemplo | slice : 1: 6}}

Date pipe

El date pipe sirve para formatear nuestras fechas
la sintaxis es la siguiente
date : formato? : zonaHoraria ? : region?

Ejemplo:

Sin pipe: {{ fechaHoy }}

Con pipe: {{ fechaHoy | date}}

Con formato: {{ fechaHoy | date : 'shortDate'}}

Personalizado: {{ fechaHoy | date : 'dd/MM/yyyy'}}

Algunos ejemplos
  • {{animal}}
  • {{animal | uppercase}}
Pipe personalizado
Los pipes personalizados son aquellos que ejecutan alguna funcion definida por el desarrollador para gestionar la data segun la necesidad.
Ejemplo

Sin pipe: {{ 'venta' }}

Con pipe: {{ 'venta' | prefijo:'pre-':5 }}

  • {{animal}}
