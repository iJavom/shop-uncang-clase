Referencias locales y Comunicacion entre Componentes

Referencias locales

Variables que definimos en nuestro template de angular. Gracias a ella podremos hacer referencia a los elementos del DOM o componentes y directivas. Podremos usar propiedades y metodos de los seleccionados. Para definirlo podemos usar el atributo #nombreReferencia

Ejemplo:

@Input (property binding)

Es un decorador que nos permite que el componente padre pueda transmitir un dato al componente hijo, funcionara como un property binding personalizado (depende del nombre de la variable)

Ejemplo:

{{ unValorDelPadre }}

@Output (event binding)

Es un decorador que nos permite que el componente hijo pueda comunicarse con el componente padre, funciona como un event binding en el padre y del mismo que el input se llamara como se defina en el hijo
